# DISPLAYING NAME ON SEVEN SEGMENT DISPLAY

## Description
This project demonstrates how to display a name or characters using a **seven-segment display**.  
It shows basic concepts of digital electronics and controlling displays.

## How It Works
- Each segment of the display is controlled via button.  
- By turning specific buttons ON or OFF, letters of a name are displayed sequentially.  
- The characters show the full name.

## Components
- Seven-segment display (common anode)  
- Resistors 470 Ohm
- Connecting wires  
- Breadboard
- Push Button

## Applications
- Learning digital electronics  
- Name/number display boards  
- Scoreboards and counters  
- Educational and hobby projects
